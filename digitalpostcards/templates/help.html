Digital Postcards API

Digital Postcards is a simple REST API allowing you to store messages basic details of a "digital postcard" provided by the project Javascript module.

Public Endpoints

@app.route('/api/1/postcard/<int:postcard_id>', methods=['GET'])

This endpoints returns the postcard details for a given postcard ID. 

@app.route('/api/1/postcard', methods=['POST'])

This endpoint creates a postcard and adds the given postcard details to the database and sends a notification to the recipient.

Endpoints requiring Basic Authentication: @auth.login_required

@app.route('/api/1/postcard/<int:postcard_id>', methods=['PUT'])

This endpoints triggers a job to send the email notification of a given postcard. 

@app.route('/api/1/postcards', methods=['GET'])

This endpoint returns all postcards in the database. 

@app.route('/api/1/postcards', methods=['DELETE'])

This endpoint deletes all postcards in the database. 

@app.route('/api/1/postcard/<int:postcard_id>', methods=['DELETE'])

This endpoint deletes a postcard for a given postcard ID.
